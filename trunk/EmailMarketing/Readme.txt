Email Marketing
---------------

(1) Cấu hình:
IDE: Visual Studio 2012
DB: SQL Server 2008
DB Demo: dbEmailMarketing.rar ==> Restore vào SQL Server

(2) Các chức năng cơ bản:
- Quản lý thông tin khách hàng.
- Quản lý danh sách mẫu email (đơn giản) chưa mail merge.
- Quản lý danh sách các sự kiện.
- Chưa kích hoạt gửi email tự động theo sự kiện.
- Phát sinh email theo sự kiện bằng tay.
- Có thể tự động gửi theo lô email dùng lệnh SendSync.
- Có thể ẩn ứng dụng dưới System Tray.
